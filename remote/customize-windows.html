Customizing Windows

Tags:

Windows can be a real pain in the ass once there is something fundamental about the way it does things that you don't like. Here are some examples on how to do things to Windows that Windows doesn't normally let you do.

  • Remove trash from the right-click "Context Menu" that appears when you right-click in the explorer: Edit Windows Explorer Context Menu
  • Remove the stupid shortcuts under "This PC" in the Explorer that are there by default, but you never press. I'm talking about the shortcuts "Documents", "Movies", "Pictures", etc: Remove Explorer ThisPC Folders.
