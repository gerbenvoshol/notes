Edit Windows Explorer Context Menu

Tags:
  • guides
  1. Use the registry editor.
  2. Navigate to HKEY_CLASSES_ROOT\Directory
  3. The shell\, shellex\ and Background\ might all contain the key you're interested in editing. Two functions you should be aware of:
    1. A present key with the name "LegacyDisable" will hide the menu item.
    2. A present key with the name "Extended" will only show when pressing Left-Shift + Right click.
